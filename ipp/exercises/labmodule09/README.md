# Programming in Python - An Introduction: Lab Module 09

### Description

Briefly describe the objectives of the Lab Module:

1) Demonstrate how to implement a simple K-Nearest Neighbors (KNN) algorithm that can perform both regression and classification. The algorithm calculates distances between data points using Euclidean and Manhattan methods, identifies the nearest neighbors, and uses them to generate predictions. The module also includes functionality to evaluate prediction accuracy using basic error metrics.

2) Demonstrate how to build a simple weather prediction system using the K-Nearest Neighbors (KNN) algorithm. The system uses historical weather data to predict temperature, humidity, and pressure values. It applies a sliding window approach to prepare training data and generates predictions based on recent observations. The module also evaluates prediction accuracy using error metrics.

3) Demonstrate how to collect, store, and manage historical weather data using a rolling buffer. The module processes incoming weather data, maintains a fixed-size history, and periodically saves data to a file. It also provides functionality to retrieve historical records and calculate basic statistics from the collected data.

4) Demonstrate how to visualize live weather data alongside predictions generated by a K-Nearest Neighbors (KNN) predictor. The module processes incoming weather data, stores actual and predicted values, and renders an animated comparison of temperature and humidity for multiple stations. The visualizer provides real-time updates and reflects the training status of the predictor.

5) Demonstrate how to implement a Python application that can predict weather using either simulated data or live weather service data. The module integrates a weather data collector, a simple KNN-based weather predictor, and a visualization component. It runs continuous prediction loops, updates predictions, and allows easy testing in simulation mode.

6) Demonstrate how to implement a simple Bubble Sort algorithm that can sort a list of numbers, strings, or other comparable elements. The algorithm repeatedly compares adjacent elements and swaps them if they are in the wrong order. The module also ensures that the original list is not modified during sorting.
